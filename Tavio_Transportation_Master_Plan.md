## Phase 4: Website Build (Week 3-5)

### 4.1 Tech Stack
**Build:** Lovable (AI-powered visual editor) and/or Cursor (code editor)
**Source control:** GitHub (Lovable auto-commits every change)
**Hosting:** Cloudflare Pages (free tier — unlimited bandwidth, fastest edge network, auto-deploys from GitHub)
**Domain:** taviotransportation.com via Namecheap
**DNS:** Cloudflare (free — point Namecheap nameservers to Cloudflare)

**Setup flow:**
1. Register taviotransportation.com on Namecheap
2. Create free Cloudflare account → add taviotransportation.com → Cloudflare gives you two nameservers
3. In Namecheap, change nameservers from default to the Cloudflare nameservers (propagation takes ~15 min to a few hours)
4. Build site in Lovable → Lovable auto-pushes code to a GitHub repo
5. In Cloudflare dashboard → Pages → "Connect to Git" → select the GitHub repo → deploy
6. In Cloudflare Pages → Custom domains → add taviotransportation.com → SSL auto-provisions
7. Every future edit in Lovable (or Cursor via GitHub) auto-deploys to Cloudflare within seconds

**Why Cloudflare Pages:**
- Free tier covers everything this site needs (unlimited bandwidth, 500 builds/month)
- Fastest edge network (~50ms globally) — critical for mobile towing searches
- Automatic SSL
- Pairs naturally with Cloudflare DNS (also free) for one unified dashboard
- Easy to add Cloudflare Web Analytics (free, privacy-friendly alternative to Google Analytics)
- If the site ever needs server-side logic, Cloudflare Workers integrates natively
